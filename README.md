# mask_face
プライバシーを守りつつ<a href=https://google.github.io/mediapipe/solutions/holistic.html>Holistic</a>を試せます.

## Environment   
* python:3.8.2  

## How to use  
1. 依存パッケージのインストール  
`pip install -r requirements.txt`
2. mask用のgif又はpngを用意  
参考サイト:https://gifmagazine.net/
3. 実行  
`python main.py --img_path=/path/to/image`
